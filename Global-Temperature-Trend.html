Global Weather Trend

This is a very simple project, introductory to data science. There are three tables: global_data containing average global temperature, city_list containing a list of cities and the countries they belong to and city_data containing average temprature of different cities. The task of this project is to plot a 7-year moving average of global temperature. Then for a choosen city, plot the 7-year moving average temperature for the chosen city. The years are not exactly aligned, and there are a few missing values.

The project files are available in in github.
